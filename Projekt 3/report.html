Webreport (in swedish)

I detta projekt satsade jag inte så stort på att komma på ett ide eller tema för min webbplats. Den är ganska lite, men alla uppgifter blev klargjorda i god tid. Målen med projektet var att införa finesser på ens egen webbplats med hjälp av verktyg. Några verktyg som jag inte kände igen var Scalable Vector Graphics (SVG), html5 elementet canvas, geolocation, jquery, media queries samt bootstrap. Planeringen av webbplatsen krävdes idéer om hur man skulle kombinera de olika verktygen med varandra. Frågor som, hur kommer jag igång? Hur mycket tid kommer detta ta? Hur får jag kompetens för att utföra uppgifterna?
1. Vad är Media Queries? Det är koder i css som gör webbplatsen mer responsiv. Uppgiften krävdes justeringar som tog ett tag, men som tur fanns det redan i bootstrap.css, vilket underlättade uppgiften.
2. En av de lättaste uppgifterna i detta projekt. Det enda man behövde göra var att ladda ner fonten från google och ersätta den med ett ny med hjälp av @font-face.
3. Uppgiften gick ut på att skapa en transformation, transition eller animering på ett objekt med hjälp av CSS-filen. Här ändrade jag backgrundsfärget till ett annat färg.
4. Fjärde uppgiften gjordes tillsammans med tredje uppgiften.
5. I början funderade jag noggrant på hur jag skulle ta mig till vägar. Jag kunde inte komma på något bra som skulle passa till min webbplats, så jag ritade bollar, som liknar planeter och färgade dom olika färger.
6. Scalable Vector Graphics (SVG) användes till att rita upp en boll i navbar, som rör sig till höger och tillbaka.
7. I denna uppgift kopierade jag java skripten från w3schools och satte sedan in den på webbplatsen.
8. Jquery fanns redan i bootstrap. Detta var ganska enkelt att implantera på sin webbplats.
9. Det första jag gjorde innan projektet var att läsa om bootstrap. Jag använde bootstrap som min baswebbplats. I början var det svårt att fatta hur bootstrap fungerade, men till sist fattade jag basen av bootstrap. Upppgiften var svårast av alla andra.
10. Projektet introducerade nya verktyg som kan vara användbar i framtiden. Jag lärde mig mest om bootstrap och jquery i detta projekt. Jag hade ganska lite tid på mig att utföra projektet. Det tog cirka 5 timmars arbete och 5 timmars läsande om bootstrap och de andra verktygen. Projektet var svårare än första och andra projektet. Om jag skulle satsat mer tid på projektet skulle jag ha gjort flera sidor och tänka ut en tema för webbplatsen.
Jag lärde mig mycket från kursen och det tackar jag för.
